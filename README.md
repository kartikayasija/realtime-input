## Tech Stack

**Client:** HTML/JS

**Server:** Node JS, Express JS, Socket.io
  

## Project initialization

Clone the project

```bash
  git clone https://github.com/kartikayasija/realtime-input
```

```bash
  ./run.sh
```

## OR

Install dependencies

```bash
  cd backend
  npm install
```

Start 

```bash
  cd backend
  npm start
  cd frontend
```
Start Live Server
